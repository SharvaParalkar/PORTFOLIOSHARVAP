Installation / Textile

Woven Pavilion

Temporary structure exploring textile logic at architectural scale using bamboo.

Physical Model • Kangaroo

This project investigates how textile logic—weaving, tension, and pattern—can be translated to an architectural scale. Using bamboo as the primary material, the pavilion explores temporary occupation and low-impact construction while maintaining a strong spatial presence.
